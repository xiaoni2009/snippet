## Python+OpenCV+Openopse 人体姿态估计

需要安装库：opencv （3.3 以上版本）

模型下载地址在 getModels.sh 文件中给出，执行命令可自动下载：

`sh getModels.sh`

windows 运行 `getModels.bat`



参考文章：[用 Python 实现抖音尬舞机](https://mp.weixin.qq.com/s/_CrlqYB1dLbQinu3dyRNcA)
