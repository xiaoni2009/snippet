## 《哪吒》票房预测

#### 运行环境
- Python 3.7

#### 运行依赖包
- requests
- matplotlib
- pandas
- numpy
- scipy

#### 代码说明
1. get_boxoffice.py 从猫眼票房获取几部参考影片的历史票房数据，结果保存在 boxoffice.csv
2. get_nezha.py 从猫眼票房获取《哪吒》票房数据，结果保存在 nezha.csv
3. plot.py 绘制历史票房走势图
4. nezha.py 对历史票房进行曲线拟合，并对《哪吒》票房进行预测
